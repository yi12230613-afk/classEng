# 🏫 창의적인 학급 자리 바꾸기 웹앱 (Class Seat Picker)

> 학생 명렬 등록부터 두근두근한 룰렛, 랜덤 애니메이션 효과까지! 
> 지루한 자리 바꾸기를 하나의 즐거운 학급 이벤트로 만들어주는 웹 애플리케이션 프로젝트입니다.

---

## 🚀 주요 기능 (Features)

### 1. 학생 명렬 관리 (Student Management)
* **텍스트/엑셀 붙여넣기:** 줄바꿈이나 쉼표로 구분된 학생 이름을 쉽게 대량 등록합니다.
* **저장 및 불러오기:** `LocalStorage`를 활용하여 브라우저를 닫아도 명렬이 유지되도록 합니다.
* **특이사항 설정 (옵션):** 시력 문제, 키 등을 고려해 '앞자리 고정' 또는 '기피 대상 지정' 같은 간단한 커스텀 규칙을 적용할 수 있습니다.

### 2. 교실 배치도 설정 (Classroom Layout)
* **바둑판 배열 설정:** 교실 구조에 맞게 가로 $N$줄 $\times$ 세로 $M$줄 크기를 자유롭게 설정합니다.
* **분단 나누기:** 통로(복도) 위치를 지정하여 1분단, 2분단 등 실제 교실과 유사하게 시각화합니다.

### 3. 창의적인 뽑기 모드 (Seat Picking Modes)
* **🎰 룰렛 돌리기 모드:** * 특정 자리를 선택하고 룰렛을 돌리면 학생 이름이 빙글빙글 돌다가 멈추며 배치됩니다.
* **⚡ 전원 동시 랜덤 모드:** * '뒤섞기' 버튼을 누르면 전 좌석의 이름표가 화려한 애니메이션과 함께 동시에 마구 섞이다가 차례대로 고정됩니다.
* **🎁 선물 상자 열기 모드:** * 모든 자리가 선물 상자 아이콘으로 가려져 있고, 교사나 학생이 자리를 클릭할 때마다 오픈되며 주인이 공개됩니다.

---

## 🛠 기술 스택 (Tech Stack)

* **Frontend:** HTML5, CSS3 (Tailwind CSS 추천), JavaScript (ES6+)
* **State Management:** LocalStorage (간이 데이터 저장용)
* **Animation:** CSS Keyframes 또는 Canvas API (룰렛 구현용)
